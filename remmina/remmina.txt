remmina - remote desktop client for GNOME desktop environment

Remmina is a remote desktop client written in GTK+, aiming to be useful for
system administrators and travellers, who need to wo in an integrated and
consistent user interface.  Currently RDP, VNC, NX, XDMCP and SSH are
suppoports multiple network protocols in an integrated and consistent user
interface. Currently RDP, VNC, NX, XDMCP and SSH are supported.

Remmina is released in separated source packages:
o "remmina", the main GTK+ application
o "remmina-plugins", a set of plugins

Remmina is free and open-source software, released under GNU GPL license.
