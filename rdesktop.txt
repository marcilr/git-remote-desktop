Packages
========
grdesktop - GNOME frontend for the rdesktop client
rdesktop - RDP client for Windows NT/2000 Terminal Server and Windows Servers


Options
=======
-k de: set keyboard layout
-g 1500x1150: set resolution of the rdesktop window
-r disk:mydisk=/home/soma: share your home directory with the remote machine

rdesktop 192.168.1.23 -k de -g 1500x1150 -r disk:mydisk=/home/soma

askubuntu.com/


Launch file
===========
o Add application/x-rdp mime entry to 
~/.local/share/applications/mimeapps.list file:

application/x-rdp=grdesktop.desktop;

archlinux.org/

o Create ~/.local/share/applications/grdesktop.desktop file.
Copied userapp-Icedove-HVUXBX.desktop to grdesktop.desktop and
updated:

[Desktop Entry]
Encoding=UTF-8
Version=1.0
Type=Application
NoDisplay=true
Exec=/usr/bin/grdesktop %u
Name=grdesktop
MimeType=application/x-rdp
Comment=grdesktop - GNOME frontend for the rdesktop client


o Verify rdp in /etc/mimetypes:

covellite:/etc$ grep rdp mime.types
